# دليل صفحة السوبر أدمن الرسمية (Portal)

تم إضافة صفحة رسمية عند تسجيل دخول **السوبر أدمن** بعنوان:

- `/super`  (بوابة السوبر)

## ماذا تظهر صفحة /super ؟

تعرض خيارين:

1) **الدخول للبرنامج**  
- إذا كان وضع الدعم (Support Mode) مفعّلًا وعلى تيننت محدد → ينتقل مباشرة إلى `/t/{tenantId}`  
- إذا كان هناك آخر مدرسة تم فتحها سابقًا → يرجع لنفس المسار  
- وإلا → يفتح صفحة السوبر أدمن `/system` لاختيار المدرسة/الدعم

2) **الدخول لصفحة السوبر أدمن**  
- ينتقل إلى `/system`

## زر الدخول/العودة بين البرنامج والسوبر

- داخل صفحات البرنامج (Tenant Area) يظهر في الهيدر (BrandedHeader) زران للسوبر أدمن:
  - **لوحة السوبر** → `/system`
  - **بوابة السوبر** → `/super`

- داخل صفحة `/system` تم إضافة زر:
  - **العودة للبرنامج**  
    يرجع لآخر صفحة داخل مدرسة تم فتحها (يتم حفظها في `sessionStorage` باسم `lastTenantPath`)  
    وإذا لا توجد مدرسة محفوظة يرجع إلى `/super`
